[W]
	Windows
[U]
	Unix


1.  [W] helpgen ()
        хелп по командам cmd
        на основе команды help

2.  [W] int/ext ()
        списки внутренних и внешних команд cmd
        на основе команды help

3.  [U] which ()
        Аналог утилиты which на bash

4.  [WU] dc (имя1 имя2)	
         сравнить два каталога как списки файлов

5.  [WU] getip ()
         все ip-адреса машины

6.  [WU] tempfile (имя)
         в заданном каталоге создать новый файл

7.  [WU] filesearch (имя маска)
         все каталоги (от заданного), в которых
         есть файл с именем, соответсвующим маске

8.  [WU] sleep_ping (число)
         подождать заданное число секунд
         использовать команду ping

9.  [W] diskfree (буква)
        свободное место на указанном диске

10. [W] assocx (имя)
        все действия по заданному расширению
        использовать команду reg

11. [WU] дано: число
         надо: найти число Фибоначчи с заданным номером

12. [WU] сосчитать количество строк в текстовом файле
        (перевод строки - CRLF/LF)

13. [W] добавить пункты в контекстные меню
        - всех файлов (открыть в блокноте)
        - директорий (открыть консоль)

14. [W] в меню File->New добавить пункт
       (3 типа действий при выборе этого пункта)

15. [W] удалить из реестра
        - значение
        - ключ
       (.reg, .bat)

16. [WU] atquake ([-r])
         если запущен quake
         закрыть quake или перезагрузить машину

--------------------------------------------------
    Доп. задачи (на 5)

1.  [WU] sizesum ()
         суммарный размер файлов на заданном диске

2.  [WU] bigfile ()
         самый большой файл на всех дисках

3.  [WU] sqr (целое число)
         квадрат заданного числа

4.  [WU] dic (имя)
         список слов с числом их вхождений
         в заданном текстовом файле

5.  [WU] autobat ()
         Скрипт, делающий себя запускаемым при
         - старте системы
         - входе пользователя
        (MAX способов)

6.  [WU] copspeed (имя)
         скорость (MB/s) копирования заданного файла

7.  [W] xxxprint ()
        список зарезервированных системой имён файлов

8.  [WU] sleep_time (число)
         подождать заданное число секунд
         использовать команду или переменную time
         (без учета перехода через сутки)
